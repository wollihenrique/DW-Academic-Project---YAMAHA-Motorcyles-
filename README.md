## 📌 Sobre o projeto

Este projeto foi desenvolvido como parte da disciplina de **Desenvolvimento Web** da faculdade.

A proposta é a criação de uma página temática sobre motocicletas da Yamaha, unindo prática acadêmica com um toque pessoal — inspirado pela conquista de adquirir minha própria **Yamaha Fazer**.

🌐 Acesse o projeto:
👉 [http://walle.infinityfree.me](http://walle.infinityfree.me)

---

## 🚀 Aprendizados

Durante o desenvolvimento deste projeto, tive a oportunidade de aprender e praticar:

* 🌐 Publicação de sites com domínio próprio
* 🎨 Estilização com CSS
* 📐 Uso de **CSS Grid** para layout
* 🧱 Estruturação de páginas com HTML semântico
* 🔄 Organização de conteúdo e layout responsivo
